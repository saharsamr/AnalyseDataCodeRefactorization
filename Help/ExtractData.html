Data Extraction for Further Analysis

This file will extract the data recieved from Eyelink and Blackrock, and store the whole needed information, with respect to the task has run, in a .mat file; which can be find in the output folder that can be set in the config file, and under subdirectories related to the task, monkey, and date. These information can be use in further analysis without the need of having the pure eyelink or blackrock's recieved data.

addpath(genpath('toolbox'));

clear all
close all
clc

warning('off', 'MATLAB:structOnObject');

dao = DAO.DAO ( ...
            CONFIG.Config.SUBJECT_NAME, ...
            CONFIG.Config.TASK_NAME, ...
            CONFIG.Config.RESEARCHER_FIRST_NAME, ...
            CONFIG.Config.RESEARCHER_LAST_NAME, ...
            CONFIG.Config.TIME_START, ...
            CONFIG.Config.TIME_END, ...
            CONFIG.Config.BAR_SAMPLING_FREQ, ...
            CONFIG.Config.EYELINK_DATA_PATH ...
        );

dao.extract_experiments_data();

warning('on', 'MATLAB:structOnObject');
